# 辅料出入仓模块
* 辅料类型枚举
    ``` JAVA
    public enum MaterialType {
        FABRIC, 
        ACCESSORY, 
        FINISH_PRODUCT
    }
    ```
* 拣料接口可重用 pickMaterial。
* 辅料确认签收接口。
    * AppStockOut/accessoryStockOut
    * 请求参数：
        ``` Javascript
            {
                "accessStockOut": [
                    {
                        "id": 1,
                        "qty1": 10.1,
                        "qty2": 10.1
                    }
                    ...
                ]
            }
        ```

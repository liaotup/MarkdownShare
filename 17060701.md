# API 补充说明
## API前缀地址封装规格示例：
``` JS
var host = 'http://192.168.0.1'
var port = 80
var projectName = 'wmsdemo'
function getPrefix () {
    return host + ':' + port + '/' + projectName + '/'
}
```
